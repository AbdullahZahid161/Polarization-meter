# Political Polarization Meter
AI-powered political speech polarization analyzer

[![GitHub Pages](https://img.shields.io/badge/Live-4285F4?style=for-the-badge&logo=github)](https://abdullahzahid161.github.io/polarization-meter)

AI-powered analyzer detecting "Us vs Them" language in political speeches.

## Features
- Real-time polarization scoring (0-100)
- "Them vs Us" ratio calculation
- Keyword highlighting
- 100% client-side (privacy-first)
- Mobile responsive
